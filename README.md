# maliang swarm

多智能体企业编排桌面应用（`Go + Wails + React`）。

## 语言版本

- [简体中文](./README.zh-CN.md)
- [English](./README.en-US.md)
- [日本語](./README.ja-JP.md)

## 快速入口

- 本地开发：`wails dev`
- 生产构建：`wails build`
- 自动发布：推送标签 `v*`（例如 `v0.1.1`）触发 GitHub Release 工作流（Windows/macOS/Linux）

## 说明

- Release 工作流文件：`.github/workflows/release.yml`
- 规划文档：`docs/swarm_enterprise_plan.md`
